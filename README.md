# Memory Protection Mechanism 🛡️
The primary objective of this project is the development of a custom-engineered memory
protection system: a low-level implementation utilizing the C++ programming language on
Linux. This project aims to translate the theoretical mechanics of stack frame layouts and
return address verification into a functional software architecture. 

In addition to protecting data, the goal is to show how separate protection modules; specifically, those detecting buffer
overflows and those mitigating Return-Oriented Programming (ROP) come together to
create a robust defense-in-depth system. 💻
